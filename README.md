
## Getting Started

```bash
cd ai-interview-mocker
npm run dev
npm run db:studio
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
Open [https://local.drizzle.studio/](https://local.drizzle.studio/) to see database tables
